# 需求文档：微信小程序

## 背景

当前系统只有 Web 管理端（`dcb-frontend/app`），面向管理员使用，功能涵盖开奖号码管理、购买记录管理、预测号码管理、用户管理、系统配置等。后端已预留微信小程序相关接口（`/api/auth/wx-login`、`wx.appid`、`wx.secret` 配置项），但仅为 mock 模式（直接接收 openid，未对接微信官方 API），且无实际小程序端代码。

需要开发微信小程序端，面向普通用户（C 端），让用户可以在手机上查看双色球开奖数据、推荐号码、个人购买记录和中奖情况。

## 目标

1. 开发微信小程序客户端，提供 C 端用户核心功能
2. 完善后端微信登录流程，对接微信官方 API（code → openid）
3. 权限体系适配：区分管理员（Web）和普通用户（小程序）的功能边界
4. 支持订阅付费（微信支付），控制普通用户的功能访问

## 功能需求

### 1. 用户登录与认证

- **微信一键登录**：调用 `wx.login()` 获取临时 code → 发送到后端 → 后端调微信 API 换 openid → 创建/查找用户 → 返回 JWT
- **自动登录**：token 有效期内免登录，token 过期后静默重新授权
- **用户信息**：获取微信昵称、头像（`wx.getUserProfile` 或新版头像昵称填写组件）

### 2. 首页 / 仪表盘

- 展示最新一期开奖号码（红球 + 蓝球）
- 展示今日推荐号码（随机或规则推荐）
- 展示个人统计数据：累计购买注数、中奖次数、总盈亏

### 3. 开奖号码查询

- 历史开奖号码列表（分页）
- 按日期范围筛选
- 单期详情：号码、和值、跨度、区间比、奇偶比、奖金分配

### 4. 推荐号码

- 系统自动推荐号码（基于冷热号、区间比、奇偶比规则）
- 支持多组号码展示
- 用户可"采纳"推荐号码转为购买记录

### 5. 我的购买记录

- 展示当前用户自己的购买记录列表
- 中奖状态标注（待开奖 / 已中奖 / 未中奖）
- 按中奖等级筛选
- 盈亏统计

### 6. 我的预测记录

- 展示当前用户自己的预测号码列表
- 命中结果展示（命中红球数、蓝球、等级）

### 7. 个人中心

- 用户信息展示（昵称、头像）
- 订阅状态与到期时间
- 订阅续费入口（微信支付）

### 8. 订阅付费（微信支付）

- 套餐展示（月卡 / 季卡 / 年卡）
- 微信支付下单
- 支付回调处理，更新用户订阅到期时间

## 非功能需求

### 性能
- 小程序首屏加载 < 2 秒
- API 响应时间 < 500ms（正常网络环境）
- 图片资源使用 CDN，列表使用分页加载

### 安全
- 所有 API 请求携带 JWT Token
- openid 不暴露给前端
- 微信支付回调验签
- 敏感配置（appid、secret、商户号）存于服务端，不可暴露到小程序端

### 兼容性
- 微信基础库版本 ≥ 2.10.0
- iOS 12+ / Android 8+
- 适配刘海屏、底部安全区

## 约束条件

### 技术栈
- **小程序框架**：原生微信小程序（WXML + WXSS + JS），不使用第三方框架（如 uni-app、Taro）
- **后端**：现有 Spring Boot 2.7.18 + MyBatis-Plus 3.5.5，新增接口在现有 Controller 扩展或新建
- **数据库**：现有 MySQL 表结构，用户表 `t_user` 已包含 openid、订阅到期时间等字段
- **前端无框架**：小程序独立开发，不依赖现有 Web 前端代码

### 现有约束
- 用户权限体系已存在（`t_user.frole`：ADMIN/USER，`PermissionHelper.checkFull`）
- 后端 `/api/auth/**` 路径不经过 JWT 拦截器
- 系统配置表已预留 `wx.appid`、`wx.secret` 配置项（当前为空）
- JWT Token 有效期通过配置表 `jwt.expire-hours` 控制
- 活跃度记录当前在登录时触发（AuthService.recordActive）

### 环境依赖
- 微信小程序 AppID（需注册微信小程序）
- 微信小程序 AppSecret
- 微信支付商户号（订阅付费需要）
- 后端服务需部署到公网可访问的服务器（微信 API 回调需要）

## 验收标准

1. 用户可通过微信一键登录小程序
2. 登录后可查看最新开奖号码和历史号码
3. 可查看系统推荐的号码方案
4. 可记录个人购买号码并查看中奖情况
5. 可记录预测号码并查看命中结果
6. 可查看订阅状态并进行续费
7. 未订阅用户的功能访问受限（如只能查看最近 10 期开奖）
8. 小程序通过微信审核上架
